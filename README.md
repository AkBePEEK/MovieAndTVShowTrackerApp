# 🎬 Movie & TV Show Tracker App

A native Android application for discovering and tracking movies, series, and watchlist favorites built with **Kotlin**.

![Kotlin](https://img.shields.io/badge/Kotlin-7F52FF?style=for-the-badge&logo=kotlin&logoColor=white)
![Android](https://img.shields.io/badge/Android-3DDC84?style=for-the-badge&logo=android&logoColor=white)

---

## 🌟 Key Features

- 🍿 **Movie & TV Discovery**: Browse trending movies, top-rated series, and upcoming releases.
- ⭐ **Personal Watchlist**: Save movies to custom favorites and track watched episodes.
- 🔍 **Detailed Insights**: View cast info, release dates, ratings, and plot overviews.
- 📱 **Android Material Design**: Smooth card layouts, animations, and dark theme UI.

---

## 🚀 Setup & Build

1. Open the project in **Android Studio**.
2. Sync Gradle files (`Gradle Sync`).
3. Run on an Android Emulator or physical device (API Level 24+).
